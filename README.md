A simple desktop-based quiz application built using **Java** and **Swing**. The program presents multiple-choice questions, enforces a timer, and automatically calculates the final score.

---

## Features

- Multiple-choice quiz questions  
- Countdown timer  
- Automatic score calculation  
- Simple graphical user interface  
- Designed using basic Object-Oriented Programming principles  

---

## Tech Stack

- Java  
- Java Swing  

---

## How to Run

1. Make sure **Java JDK 8 or higher** is installed  
2. Clone or download the repository  
3. Compile and run the main file:
   ```bash
   javac QuizApp.java
   java QuizApp
   ```
